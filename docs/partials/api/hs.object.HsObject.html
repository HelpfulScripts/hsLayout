HsObject
service in module hs.object

Description

Base class for all HelpfulScripts objects. Can be used with or without new. 

var obj1 = HsObject('<HsDerivedObject>');
var obj2 = new HsObject('<HsDerivedObject>');    // identical result

Usage

HsObject(type[, name][, parent]);

Parameters

ParamTypeDetails
typestring

the object type or category

name
(optional)
string

the object name

parent
(optional)
HsObject

If specified, adds this HsObject as a child to the parent.

Methods

  • children(child, remove)

    retrieves the list of children, or sets or removes a specific child to this parent. In addition to maintaining an array of the children to this parent, this function also generates sub-arrays of children for each Object type.

    Parameters
    ParamTypeDetails
    child
    (optional)
    		HsObjectString

    object or type string.

    HsObject: `child` will be added as a child to this parent and the child returned.
string: returns an array of all children of the specified type.
Unspecified: returns an array of all current children
    remove
    (optional)
    		Boolean

    if specified and true-ish, the specified child object will be removed from the list of children. If child is a type string, remove will have no effect.

    Returns
    object

    the child if a child HsObject was specified;
    or an array of children of a certain type, if a type string was specified;
    or the array of children if no parameter was specified.
