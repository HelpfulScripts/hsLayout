HsObject
service in module hs.object

Description

Base class for all HelpfulScripts objects. Establishes an array of children to allow for a tree structure of descendents; Can be used with or without new. 

var obj1 = HsObject();
var obj2 = new HsObject();	// identical result

Methods

  • children(child, remove)

    retrieves the list of children, or sets or removes a specific child to this parent. In addition to maintaining an array of the children to this parent, this function also generates sub-arrays of children for each Object type.

    Parameters
    ParamTypeDetails
    child
    (optional)
    		HsObjectString

    String: the parameter specifies an HsObject type and the function returns an array of children of this type.
    HsObject: child will be added as a child to this parent and the parent returned.
    Unspecified: the list of current children will be returned as an Array.

    remove
    (optional)
    		Boolean

    if specified and true-ish, the specified child object will be removed from the list of children. If child is a type string, remove will have no effect.

    Returns
    object

    this parent if a child HsObject was specified, to allow for chained calls;
    or an array of children of a certain type, if s type string was specified;
    or the array of children if no parameter was specified.
