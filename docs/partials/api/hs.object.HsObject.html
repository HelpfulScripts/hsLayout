HsObject
service in module hs.object

Description

Base class for all HelpfulScripts objects. Can be used with or without new. 

var obj1 = HsObject();
var obj2 = new HsObject();	// identical result

Usage

HsObject(type, name[, parent]);

Parameters

ParamTypeDetails
typestring

the object type or category

namestring

the object name

parent
(optional)
HsObject

If specified, adds this HsObject as a child to the parent.
