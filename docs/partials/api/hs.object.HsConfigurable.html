HsConfigurable
service in module hs.object

Description

provides for a basic configuration mechanism and establishes an array of children to allow for a tree structure of descendents;

Methods

  • cfg(d, cb)

    Configuration function that doubles as a configuration object. This object consists of a list of key/value pairs in object literal notation: 

        { key1: value1, key2: value2, ...}

    Object Literal configuration

    Calling cfg(d) with an object literal adds or replaces the provided keys in the configuration object. Calling cfg() without a parameter returns the configuration object.

    Parameters
    ParamTypeDetails
    d
    (optional)
    		Object

    specifies a JSON file or an Object literal containing key value pairs to add or replace in the cfg object.

    cb
    (optional)
    		function

    a callback function to call when loading of the configuration is completed.

    Returns
    HsObject

    the configuration HsObject when called without parameter, otherwise the HsObject itself. This allows chaining of method calls during setup.

    children(child, remove)
    

    retrieves the list of children, or sets or removes a specific child to this parent.
In addition to maintaining an array of the children to this parent, this function also
generates sub-arrays of children for each Object type.
    

    Parameters
    ParamTypeDetails
    child 
    (optional)
    		HsObjectString
    String: the parameter specifies an HsObject type and
the function returns an array of children of this type.

    HsObject: child will be added as a child to this parent and the child 
returned.

    Unspecified: the list of current children will be returned as an Array. 
    

    remove 
    (optional)
    		Boolean
    if specified and true-ish, the specified child object will be removed
from the list of children. If child is a type string, remove will have no effect.
    

    Returns
    object
    this parent if a child HsObject was specified, to allow for chained calls;

    or an array of children of a certain type, if s type string was specified;

    or the array of children if no parameter was specified.
