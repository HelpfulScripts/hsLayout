hsMoveableCtrl
controller in module hsWidgets

Description

Controller to manage move and size events on widgets.

Methods

  • moveable(elem)

    Parameters
    ParamTypeDetails
    elemjQuery selection

    the element for which hs-moveable was defined

Events

  • mouseDown

    defines an move, size, or full-screen event-in-progress and shows the helper frame.

  • mouseMove

    if an event is in progress, then adjusts the position or size of the widget

  • mouseUp

    hides the helper frame and clears the move in progress. Also, the time of latest mouseUp is used to determine a double click
