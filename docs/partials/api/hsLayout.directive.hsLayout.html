hsLayout
directive in module hsLayout

Description

hs-layout directive. Establishes a space in which widgets live.

Attributes

All attributes are optional except where marked as required. For emphasis, optional attributes are wrapped in square brackets: [...]. When optional, the default value is underlined.

  • [hs-type = 'tiles | columns | rows'] sets the type of layout. For options see below
  • [hs-tiles] create a tile layout
  • [hs-columns='[Array, ]'] Please see columns layout on avaliable options for Array
  • [hs-rows='[Array, ]'] Please see rows layout on avaliable options for Array
  • [hs-fill-last-col] applies to tile layout only; if specified, the last colum of tiles are stretched horizontally to fill the remaining space.

Usage

as element:
<hs-layout>
</hs-layout>
