HsTileLayout
service in module hsLayout.object

Description

Extends HsLayout. Tile layout fits widgets into equal sized tiles from left to right, and top to bottom. Tiles are arranged in square-like fashion that extend first horizontally, then vertically. The width of the last widget in each row is set to fit the remnaining container width.
