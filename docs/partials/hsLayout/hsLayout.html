#Angular Widget and Layout Library Provides AngularJS directives to facilitate widgets and layouts.

Layouts can be manual or automatic. Available automatic layouts include row layouts, columns layouts, and tiles layouts.

Layouts can be nested - see example below. The top level layout will try to maximize its size within the constraints of the surrounding container. For example, as first element with non-static position, a layout will fill and make available the entire browser window. This enables treating the browser window as an application gui on which widgets (views, controls, etc.) can be placed in deliberate locations.

Widgets live inside a layout. Double clicking the widgets will reversibly toggle its full-screen mode display. Add the hs-moveable attribute to a widget to allow it to be manually moved or resized. The codebase is available at https://github.com/HelpfulScripts/hsLayout.
