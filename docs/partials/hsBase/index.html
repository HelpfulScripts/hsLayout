HelpfulScripts foundational services

Provides

  • hs: a simple service registry and dependency injection
  • hsDeprecated: a simple deprecation message service
  • hsBase.object: a simple object inheritance model
  • HsObject: Helpful Scripts base object
  • HsConfigurable: A base object with configuration options

Dependencies

  • AngularJS 1.5.0 or higher; optional, for integration with angular modules
