HsObject
service in module hs.object

Description

Base class for all HelpfulScripts objects. Can be used with or without new. 

var HsObject = hs('HsObject');
var Person = new HsObject('Person');
var parent = new Person('John');
var child  = HsObject('Person', 'Jane', parent);
Objects derived from HsObject can be called as a function. Override _outerFunction to customize behavior. See hs.object overview for an example.

  • children(child, remove)

    • #.children() retrieves the list of children,
    • #.children(child) adds a child to this parent and returns the child.
    • #.children(child, true) removes a child from the parent and returns the child.
    • #.children(type) returns all children of the specified type.
    Parameters
    ParamTypeDetails

    HsObject: `child` will be added as a child to this parent and the child returned.
string: returns an array of all children of the specified type.
Unspecified: returns an array of all current children
    remove
    (optional)
    		Boolean

    if specified and true-ish, the specified child object will be removed from the list of children. If child is a type string, remove will have no effect.
