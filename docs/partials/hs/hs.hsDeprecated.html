hsDeprecated
service in module hs

Description

a utility component to mark deprecated pieces of code. The default implementation prints a stack trace to indicate where a deprecated call occured.

 var hsDeprected = hs('hsDeprecated'); // inject hsDeprecated
 var deprecated = new hsDeprecated();  // create new function with default reporting
 ...
 function foo() {
     deprecated('please use "bar()" instead'); // generate console message
 }

will produce a console message followed by a stack trace.

Usage

hsDeprecated([printMessage][, printTrace]);

Parameters

ParamTypeDetails
printMessage
(optional)
boolean

(default: true) boolean to indicate whether or not to print the message.

printTrace
(optional)
boolean

(default: true) boolean to indicate whether or not to print the stack trace.
