HsPromise
service in module hs.object

Description

implements a Promise mechanism following the template published at https://www.promisejs.org/

Methods

  • done(onFulfilled, onRejected)

    registers both handler functions for execution once the Promise has been fulfilled. The function always completes immediately and returns nothing. The appropriate handler function is always called asynchronously.

    Parameters
    ParamTypeDetails
    onFulfilledFunction

    called once Promise has been fulfilled

    onRejectedFunction

    called once Promise has been rejected

  • then(onFulfilled, onRejected)

    provides handlers to call upon fulfillment or rejection of the promise. then() works similarly to done() but in addition returns a new promise on the two provided functions. This allows for convenient chaining of promises.

    readFile(url).then(JSON.parse).then(processJSON);
    Parameters
    ParamTypeDetails
    onFulfilled
    (optional)
    		Function

    called when Promise has been fulfilled

    onRejected
    (optional)
    		Function

    called when Promise has been rejected
