  17 * 

  18 * hsLayout provides means to layout the browser window in various ways.

  19 * See [Layout Examples](example/layout.html)

  20 * 

  21 * ## Concepts

  22 * 

  23 * ### Layouts

  24 * Layouts can be either defined generically:

  25 * ```

  26 * m(Layout, {

  27 *     css: '.myLayoutClass',  // optional

  28 *     columns: []

  29 *     content: ['left', 'right']

  30 * })

  31 * ```

  32 * Or, for more complex cases, by defining a class that extends {@link view.Layout.Layout `Layout`}:

  33 * ```

  34 * class Columns extends Layout {

  35 *     getComponents(node:Vnode) {

  36 *         return [m(LeftColumn), m(CenterColumn), m(RightColumn)]

  37 *     }

  38 * }

  39 * m(Columns);

  40 * ```

  41 * 

  42 * ### Layouters

  43 * To create new layout styles, define a class that extends the abstract {@link view.Layouter.Layouter `Layouter`} class.

  44 * This class should implement the `getStyles` method which calculates the styles attributes required for each `Component`

  45 * to be layed out in a `Layout`.

  46 * 

  47 * Currently defined Layouters:

  48 * -   {@link view.PillaredLayouter.Columns `Columns`}

  49 * -   {@link view.PillaredLayouter.Rows    `Rows`}

  50 * -   {@link view.TileLayouter             `Tiles`}

  51 * 

  52 * ### Example

  53 * 

  54 * 'script.js'>

  55 * const theContent = ['Top row: 50px', 'Bottom row: remainder']

  56 * m.mount(root, {view: () => m(hsLayout.Layout, {

  57 *     css: 'myColumn',

  58 *     rows: ["50px", "fill"], 

  59 *     content:theContent

  60 *     })

  61 * });

  62 * 

  63 * 

  64 * 

  65 * ### Nested Example

  66 * 

  67 * 'script.js'>

  68 * m.mount(root, {view: () => 

  69 *     m(hsLayout.Layout, {

  70 *         css: 'myColumn',

  71 *         rows: ["150px", "fill"], 

  72 *         content:[

  73 *             m(hsLayout.Layout, {columns:['20%'], content:['top left', 'top 2nd']}), 

  74 *             m(hsLayout.Layout, {columns:['20%'], content:['bottom left', 'bottom 2nd']})

  75 *         ]

  76 *     })

  77 * });
