   2 * # hsLayout 

   3 * 

   4 * Helpful Scripts UI layout tools. 

   5 * [`[Github page]`](https://github.com/HelpfulScripts/hsLayout)

   6 * [`[Coverage Info]`](./data/src/hsLayout/coverage/)

   7 * ___

   8 * 

   9 * hsLayout provides means to layout the browser window in various ways.

  10 * See [Layout Examples](example/layout.html)

  11 * 

  12 * ## Concepts

  13 * 

  14 * ### Layouts

  15 * Layouts can be either defined generically:

  16 * ```

  17 * m(Layout, {

  18 *     css: '.myLayoutClass',  // optional

  19 *     columns: []

  20 *     content: ['left', 'right']

  21 * })

  22 * ```

  23 * Or, for more complex cases, by defining a class that extends {@link view.Layout.Layout `Layout`}:

  24 * ```

  25 * class Columns extends Layout {

  26 *     getComponents(node:Vnode) {

  27 *         return [m(LeftColumn), m(CenterColumn), m(RightColumn)]

  28 *     }

  29 * }

  30 * m(Columns);

  31 * ```

  32 * 

  33 * ### Layouters

  34 * To create new layout styles, define a class that extends the abstract {@link view.Layouter.Layouter `Layouter`} class.

  35 * This class should implement the `getStyles` method which calculates the styles attributes required for each `Component`

  36 * to be layed out in a `Layout`.

  37 * 

  38 * Currently defined Layouters:

  39 * -   {@link view.PillaredLayouter.Columns `Columns`}

  40 * -   {@link view.PillaredLayouter.Rows    `Rows`}

  41 * -   {@link view.TileLayouter             `Tiles`}

  42 * 

  43 * ### Example

  44 * 

  45 * 'script.js'>

  46 * const theContent = ['Top row: 50px', 'Bottom row: remainder']

  47 * m.mount(root, {view: () => m(hsLayout.Layout, {

  48 *     css: 'myColumn',

  49 *     rows: ["50px", "fill"], 

  50 *     content:theContent

  51 *     })

  52 * });

  53 * 

  54 * 

  55 * 

  56 * ### Nested Example

  57 * 

  58 * 'script.js'>

  59 * m.mount(root, {view: () => 

  60 *     m(hsLayout.Layout, {

  61 *         css: 'myColumn',

  62 *         rows: ["150px", "fill"], 

  63 *         content:[

  64 *             m(hsLayout.Layout, {columns:['20%'], content:['top left', 'top 2nd']}), 

  65 *             m(hsLayout.Layout, {columns:['20%'], content:['bottom left', 'bottom 2nd']})

  66 *         ]

  67 *     })

  68 * });

  69 *
