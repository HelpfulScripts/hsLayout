   2 * # hsLayout 
   3 * 
   4 * Helpful Scripts UI layout tools. [Github page](https://github.com/HelpfulScripts/hsLayout)
   5 * 
   6 * hsLayout provides means to layout the browser window in various ways.
   7 * See [Layout Examples](example/layout.html)
   8 * 
   9 * ## Concepts
  10 * 
  11 * ### Layouts
  12 * Layouts can be either defined generically:
  13 * ```
  14 * m(Layout, {
  15 *     css: '.myLayoutClass',  // optional
  16 *     columns: []
  17 *     content: ['left', 'right']
  18 * })
  19 * ```
  20 * Or, for more complex cases, by defining a class that extends {@link Layout.Layout `Layout`}:
  21 * ```
  22 * class Columns extends Layout {
  23 *     getComponents(node:Vnode) {
  24 *         return [m(LeftColumn), m(CenterColumn), m(RightColumn)]
  25 *     }
  26 * }
  27 * m(Columns);
  28 * ```
  29 * 
  30 * ### Layouters
  31 * To create new layout styles, define a class that extends the abstract {@link Layouter.Layouter `Layouter`} class.
  32 * This class should implement the `getStyles` method which calculates the styles attributes required for each `Component`
  33 * to be layed out in a `Layout`.
  34 * 
  35 * Currently defined Layouters:
  36 * -   {@link PillaredLayouter.Columns `Columns`}
  37 * -   {@link PillaredLayouter.Rows    `Rows`}
  38 * -   {@link TileLayouter             `Tiles`}
  39 * 
  40 * ### Example
  41 * 
  42 * 
  43 * const theContent = ['Top row: 50px', 'Bottom row: remainder']
  44 * m.mount(root, {view: () => m(hslayout.Layout, {
  45 *     css: 'myColumn',
  46 *     rows: ["50px", "fill"], 
  47 *     content:theContent
  48 *     })
  49 * });
  50 * 
  51 * 
  52 * 
  53 * ### Nested Example
  54 * 
  55 * 
  56 * m.mount(root, {view: () =>

  59 *         rows: ["150px", "fill"], 
  60 *         content:[

  62 *             m(hslayout.Layout, {columns:['20%'], content:['bottom left', 'bottom 2nd']})
  63 *         ]
  64 *     })
  65 * });
