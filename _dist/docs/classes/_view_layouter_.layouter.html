Class Layouter

Abstract base class for creating layout style implementations. Subclasses should implement getStyles. In addition, subclasses need to be registered with the static Layouter.register method.

Example

class MyLayouter extends Layouter { cssClass = '.my-css-class'; constructor(public areaDesc:LayoutToken[]) { super(areaDesc); }

protected getStyles(components:Array):string { components.map((c:Layout|Vnode, i:number) => { c.style = `width:auto; height:auto; }); return this.cssClass; } }

Layouter.register('MyLayouter', MyLayouter);
